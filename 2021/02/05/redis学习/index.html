redis 简介

redis 是一种非关系型数据库（NoSQL），key-value 数据库，内存数据库，支持持久化，事务和备份，集群（支持16个库），等高可用功能，并且性能高（可达到每秒 100000+ 次查询），易扩展，丰富的数据类型，所有操作都是单线程，原子性
